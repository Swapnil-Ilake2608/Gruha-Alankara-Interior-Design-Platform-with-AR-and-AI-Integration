# Demo Script (2-3 Minutes)

## 0:00 - 0:20 Intro
- Open Home page.
- Explain: AI-powered interior design with AR preview and Buddy auto-booking.

## 0:20 - 0:55 Analyze Room
- Go to Analyze Room.
- Pick style + language.
- Upload a room image or start camera and capture.
- Show analysis spinner and transition to Design Studio.

## 0:55 - 1:40 Design Studio + Buddy
- Show color palette and furniture recommendations.
- Open Buddy AI chat.
- Click "show recommendations".
- Click "book now".
- Highlight Buddy confirmation message with item prices.

## 1:40 - 2:10 AR Preview
- Click Live AR Camera.
- Select furniture item chips.
- Show stable object preview and placement controls.

## 2:10 - 2:40 Profile + History
- Click user name -> Profile.
- Show booking history with status and prices.
- Mention persistence in SQLite across logins.

## 2:40 - 3:00 QA + Close
- Mention health endpoint, smoke tests, and CI workflow.
- Close with completion percentage and optional enhancement path.
